## 🧠 The Big Idea

Over **250 million Indians** use feature phones and are completely excluded from digital payments like UPI — not because they can't pay, but because the system wasn't built for them.

This project changes that.

I'm building a **lightweight voice biometric authentication system** that works on basic cellular networks — no smartphone needed. Think secure UPI-like transactions through just your voice, even on a ₹500 phone.

This repository documents every step of that journey — the learning, the building, the failures, and the breakthroughs.

## 💡 Why This Project?

Most ML portfolios are filled with the same Titanic survival and house price projects.

This is different because:
- 🇮🇳 It solves a **real Indian problem** nobody else is working on
- 📱 It targets **feature phone users** — the most underserved segment
- 🧠 It combines **audio ML + fraud detection + Edge AI** in one system
- 📡 It works under **real-world constraints** — low bandwidth, basic hardware
- 🔒 It tackles **biometric security** — a growing field with huge demand
